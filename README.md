# snippets

[![GitHub stars](https://img.shields.io/github/stars/xu-haozhe/snippets?style=flat-square)](https://github.com/xu-haozhe/snippets/stargazers)

各种零零散散的小玩意，没准哪天对你有帮助，有用的话给个 ⭐star 吧

## 怎么用？

克隆或是下载，通常我会在子目录里留 readme 的。

有的需要自己编译，有的需要配置环境。我会尽量把依赖写清楚的。

## 跑不起来？

用你精湛的技术修复它，成功了开个 [PR](/compare) 吧

问问 AI，他们也许能给你帮助

整一个 [Issues](https://github.com/xu-haozhe/snippets/issues/new)  试试。不过没准我也不知道怎么让它在你的电脑上跑起来，毕竟可能我都忘了😂

## 参与贡献

整一个 [issue](https://github.com/xu-haozhe/snippets/issues/new) 试试，有时间没准就改了

**强烈推荐**：整一个 [PR](https://github.com/xu-haozhe/snippets/compare) ，我会非常感激的，因为不用我自己改😂
