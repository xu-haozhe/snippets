# txt2lrc

### 轻松转换txt在音频播放器中阅读

使用此脚本将txt文件转成lrc与wav文件，以供放入音频播放器阅读（并没有AI配音功能）。

~~通常用来在学习机或者听力播放器上看小说~~

## 依赖

+ python3.12.3+


## 使用方法

```shell
python txt2lrc.py [-h] -i INPUT [-o OUTPUT] [--title]
``` 

参数说明：

+ -i, --input INPUT 输入文件/目录路径

+ -o, --output OUTPUT 输出文件/目录路径

+ --title 是否将标题添加到歌词中，在文件中添加如下开头：

```lrc
{filenameNoExt}
----------------------------------------
```

+ -h, --help 显示帮助信息

或者，你也可以用 GUI 版本，需要 pyqt6，但是我也忘了怎么安装 pyqt6 了

## 注意

此脚本仅在 linux 下测试过，其他平台可能会出现问题

仅支持 utf-8 ,请确认文件编码。对于非 utf-8 编码的文件，脚本行为未定义

别放太长的文本，一个文件放一章没问题，太多可能会出错

空格或空行可能被某些设备忽略
